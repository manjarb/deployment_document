# Installing Node.js

**Installing Node.js via package manager**

Login to the server
```sh
local $ ssh user@your_server_ip
```

Debian and Ubuntu based Linux distributions
```sh
$ curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```
